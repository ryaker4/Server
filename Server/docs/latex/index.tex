\hypertarget{index_intro}{}\doxysection{Введение}\label{index_intro}
Сервер обработки векторов — это многокомпонентное приложение для аутентификации клиентов и обработки числовых векторных данных. Сервер поддерживает протокол аутентификации на основе SHA224 и предоставляет функционал для суммирования векторов с контролем переполнения.\hypertarget{index_features}{}\doxysection{Основные возможности}\label{index_features}

\begin{DoxyItemize}
\item Аутентификация клиентов с использованием хэширования SHA224 с солью
\item Обработка векторных данных с контролем переполнения
\item Многопоточное логирование в файл
\item Конфигурация через командную строку
\item Загрузка базы данных клиентов из файла
\end{DoxyItemize}\hypertarget{index_architecture}{}\doxysection{Архитектура системы}\label{index_architecture}
